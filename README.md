# Business Intelligence and Sales Trend Analysis

## Project Overview

This project focuses on analysing retail product sales data to discover meaningful business insights related to customer behaviour, product profitability, customer loyalty, and seasonal sales trends. The analysis was performed using Python-based data analytics and visualisation techniques to support data-driven business decision-making.

The project includes data cleaning, exploratory data analysis (EDA), descriptive statistical analysis, customer segmentation using RFM analysis, profitability analysis, correlation analysis, and sales trend visualisation.

# Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Statistical analysis using mean, median, standard deviation, quartiles, and skewness
* Customer loyalty analysis using RFM (Recency, Frequency, Monetary)
* Product category profitability analysis
* Monthly and seasonal sales trend analysis
* Correlation matrix analysis
* Business data visualisation using Python libraries

# Technologies Used

# Dataset Information

The dataset contains customer-level retail transaction data including:

* Order details
* Customer information
* Product categories
* Sales revenue
* Quantity purchased
* Discounts
* Profit values


# Project Workflow

## 1. Data Cleaning and Processing

* Imported and loaded the dataset using Pandas
* Checked missing values and duplicates
* Processed and cleaned the dataset for analysis

## 2. Exploratory Data Analysis (EDA)

Performed detailed statistical analysis including:

* Mean Analysis
* Median Analysis
* Standard Deviation Analysis
* Quartile Analysis
* Skewness Analysis
* Correlation Analysis

## 3. Customer Loyalty Analysis

Implemented RFM analysis to identify:

* High-value customers
* Loyal customer segments
* Customer purchase behaviour

## 4. Product Profitability Analysis

Analysed:

* Revenue generated by each product category
* Profitability trends
* Impact of discounts on profit

## 5. Sales Trend Analysis

* Monthly sales trend analysis
* Seasonal business performance analysis
* Peak sales month identification

# Key Insights

* Technology products generated the highest revenue and profit.
* Furniture category showed high sales but lower profitability.
* Excessive discounts negatively affected profit margins.
* A small group of loyal customers contributed significantly to total revenue.
* Sales performance showed strong seasonal trends, especially during Q4.
